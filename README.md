# HTTP-Storm
HTTP Storm é um servidor HTTP/1.1 assíncrono de alta velocidade em C++20, processando >980.000 req/s em benchmarks. Usa epoll Linux para I/O O(1), latência &lt;1ms e escalabilidade massiva. Inclui TLS 1.3, middleware avançado (Gzip, CORS, Tracing), roteamento dinâmico e segurança integrada, ideal para infraestrutura crítica.
